# 🛢️ SQL Programming Course (Εμπλουτισμένη Έκδοση)

Καλώς ήρθατε στο αποθετήριο μαθημάτων SQL! Αυτό το αποθετήριο περιέχει ένα πλήρες, δομημένο εκπαιδευτικό πρόγραμμα (curriculum) για την εκμάθηση της γλώσσας ερωτημάτων SQL.

Όλα τα μαθήματα είναι γραμμένα στα Ελληνικά και σε μορφή **Jupyter Notebooks**, ώστε να μπορείτε να διαβάζετε τη θεωρία και να τρέχετε τον κώδικα (μέσω SQLite) ταυτόχρονα!

> 💡 **Νέο!** Σε κάθε ενότητα περιλαμβάνονται **Ασκήσεις** (`Exercises.ipynb`) και **Λύσεις** (`Solutions.ipynb`) για εξάσκηση!

---

## 📚 Δομή Μαθημάτων (Curriculum)

1. **[01_SQL_Basics](01_SQL_Basics)**: Εισαγωγή, SELECT, FROM, WHERE, ORDER BY.
2. **[02_Data_Manipulation](02_Data_Manipulation)**: INSERT, UPDATE, DELETE (DML).
3. **[03_Data_Definition](03_Data_Definition)**: CREATE, ALTER, DROP (DDL).
4. **[04_Functions_and_Aggregation](04_Functions_and_Aggregation)**: GROUP BY, HAVING, Συναρτήσεις συνάθροισης.
5. **[05_Joins_and_Relations](05_Joins_and_Relations)**: INNER, LEFT, RIGHT JOIN.
6. **[06_Advanced_Querying](06_Advanced_Querying)**: Subqueries, CTEs, Set Operations.
7. **[07_Window_Functions](07_Window_Functions)**: OVER(), PARTITION BY, RANK().
8. **[08_Database_Objects](08_Database_Objects)**: Views, Indexes.
9. **[09_Transactions_and_Security](09_Transactions_and_Security)**: ACID, COMMIT, ROLLBACK.
10. **[10_Projects](10_Projects)**: Ολοκληρωμένα Mini-Projects.

---

## 🚀 Πώς να τρέξετε τα μαθήματα

1. Κατεβάστε το αποθετήριο.
2. Εγκαταστήστε το Jupyter και βιβλιοθήκες: `pip install jupyter pandas sqlite3` (αν και το sqlite3 είναι built-in στην Python).
3. Ανοίξτε το τερματικό και τρέξτε: `jupyter notebook`
